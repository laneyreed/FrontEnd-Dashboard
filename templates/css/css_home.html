CSS

  • stands for Cascading Style Sheets
  • defines how HTML elements should be displayed on a web page
  • controls aspects such as layout, typography, visuals and responsiveness

Fundamentals

CSS fundamentals encompass the core concepts and techniques required to effectively style web pages

Syntax

Understanding the structure of CSS rules, which consist of selectors, properties, and values.

  • Selectors: various ways to target specific HTML elements for styling, including element type selectors, class selectors, and ID selectors
  • Properties: styling properties such as colors, typography and dimensions
  • Values:
CSS
selector{
  property: value;
